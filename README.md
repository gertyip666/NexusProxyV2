## ⚡ Возможности

- ✅ MTProto + WebSocket прокси
- ✅ SOCKS5 сервер на телефоне
- ✅ Обход блокировок Telegram
- ✅ Выбор порта (1080, 8080, 8888)
- ✅ Анимированный переливающийся логотип
- ✅ Загрузочный экран с прогресс-баром
- ✅ Фоновая работа через Foreground Service
- ✅ Статистика трафика и скорости
- ✅ Журнал событий
- ✅ Копирование настроек в буфер
- ✅ Быстрые действия (сброс статистики, поделиться, открыть Telegram)

## ⚙️ Настройка Telegram

1. Открой приложение → **ЗАПУСТИТЬ**
2. Telegram → Настройки → Данные и память → Прокси
3. Тип: **SOCKS5**
4. Сервер: `127.0.0.1`
5. Порт: `1080` (или 8080, 8888)
6. Включи
